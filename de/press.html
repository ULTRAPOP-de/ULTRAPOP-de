Kurze Beschreibung

Crossfoam analysiert und visualisiert Ihre Filterblasen in sozialen Medien (bisher nur Twitter), so dass Sie feststellen können, inwieweit Ihre Online-Community eine Echo-Kammer oder ein Ort von Meinungsvielfalt ist. Eine detaillierte Analyse ermöglicht es Ihnen, Ihr soziales Netzwerk zu erforschen, es zu diversifizieren oder - wenn Sie es vorziehen - tiefer in Ihre Blase einzudringen. Crossfoam wird so zu einem Werkzeug der digitalen Selbstbestimmung.

Um diese Erfahrung zu ermöglichen, können Sie mit Crossfoam Ihr eigenes Social-Media-Konto (oder das einer anderen Benutzer*in) auswählen und das Netzwerk dieses Kontos auf Ihren lokalen Computer herunterladen. Dann wird eine Netzwerkanalyse durchgeführt, die versucht, Cluster innerhalb Ihres Netzwerks (Blasen) zu identifizieren. Sie können dann das eingebaute Visualisierungssystem verwenden oder die Daten zur Verwendung in einer anderen Anwendung herunterladen.

Crossfoam wurden mit einem starken Fokus auf Privatsphäre entwickelt. Alle Daten, Analysen und Visualisierungen werden auf Ihrem lokalen Rechner durchgeführt, es sind keine externen Dienste involviert.
